# Form Nhận Source Code — Nội Dung Mẫu

> Copy nội dung này vào Google Form (https://forms.google.com)

---

## Tiêu đề Form:
**Nhận Source Code LMS Platform**

## Mô tả:
Chào bạn! Sau khi hoàn tất đăng ký khóa học, hãy điền form này để nhận
quyền truy cập source code xây dựng website riêng.

Bạn cần có tài khoản GitHub (miễn phí) trước khi điền form.
Nếu chưa có, đăng ký tại: https://github.com/signup

---

## Các trường trong form:

### 1. Họ và tên đầy đủ
- Loại: Short answer
- Bắt buộc: Có

### 2. Email đăng ký khóa học
- Loại: Short answer (Email validation)
- Bắt buộc: Có

### 3. Mã đơn hàng (Order Code)
- Loại: Short answer
- Bắt buộc: Có
- Mô tả: "Mã đơn hàng bạn nhận được sau khi thanh toán (vd: DKxxxxxxx)"

### 4. GitHub Username
- Loại: Short answer
- Bắt buộc: Có
- Mô tả: "Username GitHub của bạn (KHÔNG phải email). 
  Xem tại: đăng nhập github.com → click avatar góc phải → dòng đầu tiên là username.
  Ví dụ: nguyenvana (không có dấu @)"

### 5. Đồng ý điều khoản sử dụng
- Loại: Checkbox
- Bắt buộc: Có
- Nội dung:

  "Tôi đã đọc và đồng ý với các điều khoản sau:
  
  ✅ Tôi chỉ sử dụng source code cho 1 website cá nhân của tôi.
  ✅ Tôi KHÔNG chia sẻ, gửi, hoặc đăng source code lên bất kỳ đâu.
  ✅ Tôi KHÔNG bán lại hoặc tạo khóa học/sản phẩm từ source code này.
  ✅ Tôi hiểu rằng vi phạm sẽ bị thu hồi quyền truy cập.
  ✅ Tôi hiểu rằng source code được cung cấp nguyên trạng (as-is)."

---

## Sau khi học viên submit form:

1. Kiểm tra mã đơn hàng (order code) trên Admin Dashboard
2. Nếu hợp lệ → invite GitHub username vào repo
3. Gửi email xác nhận cho học viên (kèm link SETUP.md)

---

## Email mẫu gửi sau khi invite:

Tiêu đề: Bạn đã được cấp quyền truy cập Source Code!

Nội dung:

Chào [TÊN],

Bạn đã được cấp quyền truy cập source code LMS Platform.

Bước tiếp theo:
1. Kiểm tra email từ GitHub → Click "Accept invitation"
2. Vào repo → Click "Fork" để tạo bản copy riêng
3. Làm theo hướng dẫn trong file SETUP.md

Link repo: https://github.com/ledangkhuong/dangkhuong-platform
Hướng dẫn chi tiết: xem file SETUP.md trong repo

Lưu ý quan trọng:
- KHÔNG chia sẻ link repo cho người khác
- KHÔNG đăng code lên nơi công cộng
- Vi phạm sẽ bị thu hồi quyền truy cập

Chúc bạn xây dựng website thành công!
